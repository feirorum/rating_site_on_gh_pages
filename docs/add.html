Add a New Item

To add a new item to the ratings hub, you'll create a new GitHub issue using our template.

How it works:

  1. Click the button below to open the GitHub issue form
  2. Fill in the item details:
    • Title: A short, descriptive name
    • Source URL: Link to the original content
    • Thumbnail URL: Image URL (optional)
    • Summary: Brief description
  3. Submit the issue
  4. Your item will appear on the homepage!
